# Project-Deteksi-Penggunaan-Helm-pada-Pengendara-Sepeda-Motor-Secara-Realtime-
Proyek ini mengimplementasikan teknik Computer Vision dan Deep Learning untuk mendeteksi apakah seorang pengendara sepeda motor mengenakan helm atau tidak. Sistem ini dirancang secara khusus untuk beroperasi secara real-time, memproses frame demi frame dari masukan video (seperti rekaman CCTV atau webcam langsung).
